# 🌐 Internet Packet Explorer (IPE)

**Live:** [ipe-playground.vercel.app](https://ipe-playground.vercel.app)

An educational playground that visualizes how messages travel across the internet as packets. Learn networking concepts interactively — packet splitting, routing, loss simulation, and reassembly.

## 🚀 Features

- **Packet Splitting** — See how messages are broken into smaller packets with sequence numbers
- **Network Journey** — Watch packets travel through router nodes with animated visualizations
- **Packet Loss Simulation** — Toggle between normal delivery and packet loss modes
- **Reassembly** — See how packets combine back into the original message
- **Bilingual** — Indonesian + English language support
- **Dark Mode** — Full light/dark theme support

## 🛠️ Tech Stack

- **Next.js 15** + TypeScript
- **Tailwind CSS 3.4** + shadcn/ui
- **Radix UI** primitives
- **Lucide** icons
- **next-themes** for dark mode

## 🏃 Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📦 Build & Deploy

```bash
npm run build
npm start
```

Deploy to Vercel with zero configuration.
